Audiost@t




Audiost@t was created for the University of Toronto and is used by several of their French language teachers.

Audiost@t is a web plateform designed for teachers that allows them to assess and grade students' phonetical skills and pronunciation in French. It allows teachers to create new exercises (new texts to read). The students can record themselves reading the text and the teacher can then tag all the pronunciation errors in the text using a correction code (with symbols, arrows, underlined letters, etc.)

This platform took several months to build. It also uses an Apache server with PHP and a MySQL database. It was built entirely from scratch.

One of the challenges of this app was to engineer a system to put the different color symbols over the text and have them change their shapes when the text is resized within a different container. It generates SVGs on the fly, depending on the position and length of the text selected.

It uses the Web Audio API to record and play sounds within the browser, and canvas to draw the waveform of the sounds recorded. There is also an MP3 encoder that takes the recorded data from the microphone and exports it into an MP3 file on the server.

A web app designed for students who are learning French and want to practice their pronunciation.
Students need to perform an audio dubbing of a movie sequence in French.

The back-end of this platform was built using an Apache server and PHP. It uses a MySQL database to store all the student information and JSON files to encode the different exercices. On the front-end side, it is built on JavaScript and uses the Web Audio API to record sounds within the browser. The text animations are made using CSS transitions. The app automatically calculates the speed of the scrolling text.

There is a web interface to create new exercises and synchronize the text with the video.
