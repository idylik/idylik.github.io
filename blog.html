Books every web developper should read at least once...

This book is meant to teach the fundamentals of computer science. It presents common algorithms and data structures. It gives a wide array of exercises and problems to solve and discusses the best way to use logic, math, and algorithms to provide the most efficient solution. Highly recommended!



In "Design for hackers", David Kadavy outlines the basic design principles when building web sites and user interfaces: color theory, proportion and geometry, size and scale, the use of white space, composition and design rules, typographic etiquette, etc.



This book is all about usability and presents the principles of intuitive navigation and information design. More than anything, it teaches common sense through real web examples. The author's humor also makes it a very fun read.
